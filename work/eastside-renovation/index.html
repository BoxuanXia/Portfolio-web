Eastside Friendly Property Renovation

Eastside Friendly Property Renovation

Interactive Web GIS application to demonstrate property renovation before/after rendering effects.

DATE
2024
TOOLS
ArcGIS Experience Builder, ArcGIS Online, 3D Rendering

Overview

This project focused on visualizing the impact of property renovations in the Eastside neighborhood. By providing a "before and after" rendering effect, stakeholders can better understand the potential of urban renewal.

Technical Details

  • Built with ArcGIS Experience Builder for a responsive web experience.
  • Implemented high-fidelity data layers to showcase rendering transitions.
  • Integrated with local real estate datasets to provide context for renovation costs.
